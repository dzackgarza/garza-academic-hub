# Tech Stack

The workspace relies on a modern TypeScript, React, and Bun ecosystem.

## Technologies

1. **Vite + React (v18)**: Core Single Page Application framework.
2. **TypeScript**: Strongly-typed language layout.
3. **Tailwind CSS (v3) + @tailwindcss/typography**: Custom design styling using curated HSL color schemes.
4. **Bun**: JavaScript runtime used for quick scripting and build pipeline execution.
5. **Pandoc**: System utility required for converting high-fidelity LaTeX equations and standard Markdown into HTML segments.
6. **MathJax**: Injected frontend math equation compiler triggered post-dynamic loading of blog post HTML segments.
7. **Playwright**: Cross-browser visual regression testing automation tool.
